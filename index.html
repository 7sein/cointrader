Cointrader

Java-based backend for algorithmically trading bitcoin and other cryptocurrencies

About Cointrader

Cointrader provides data collection and export, complex event processing and triggering, and backtesting - paper trading - live trading.

Feature Highlights

  • Connectivity to many exchanges
  • Control console (text)
  • Basic order routing
  • Simulated trading
  • Schema and persistence
  • CSV output of market data
  • Ad-hoc table reports
  • Modular infrastructure

Coming Soon
